### Features

1. Draw FSMs, PDAs, TMs etc.
2. Convert between equivalent representations.
3. Minimize / normalize etc
4. Operations - union, intersection, kleene star etc
5. Solve computational problems like EQDFA, EDFA etc where possible
6. Export in various forms - link, image, formal definition, code etc
7. Import from various forms
8. Test against a test suite, visualize non determinism
9. Maybe a DSL to specify these machines instead of drawing them?
10. If on ipad etc, hand-draw and convert?
11. compatible with flap.js?
